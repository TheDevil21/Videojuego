# Videojuego
Funcionamiento
Para poder replicar el funcionamiento del videojuego se deben seguir los siguientes pasos:
1. Crear un nuevo archivo en Visual Studio Code, en el apartado de PlatformIO con el microcontrolador ATMEGA328P utilizando el framework de Arduino.
2. Copiar el código que se encuentra en el archivo "main.cpp" dentro de la carpeta ATmega-328P en el github y pegarlo en el archivo de VSC.
3. Crear un nuevo archivo en "mikroC PRO for PIC", con el microcontrolador PIC16F887.
4. Copiar el código que se encuentra en el archivo "MiCorazonEncantado.C", dentro de la carpeta "PIC16F887" y pegarlo en el archivo creado de mikroC PRO for PIC.
5. Descargar el archivo "Tarea_2_final.pdsprj" que se encuentra en la carpeta Proteus. O replicar las conexiones del diagrama siguiente:<img width="886" height="503" alt="image" src="https://github.com/user-attachments/assets/e8f48caa-8513-4eba-ad52-4e78abee54be" />
6. Ejecutar los códigos copiados en VSC y mikroC PRO for PIC y vincular los archivos ".hex" a cada microcontrolador correspondiente.
7. Simular el diagrama de Proteus.
